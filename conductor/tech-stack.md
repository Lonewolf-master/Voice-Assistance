# Tech Stack: Jarvis\n\n## Core Language\n- **Python 3.10+**: Chosen for its extensive libraries and ease of integration with AI/ML models.\n\n## Speech-to-Text (STT)\n- **OpenAI Whisper (Local)**: High-accuracy, transformer-based model for transcribing user speech entirely offline.\n\n## Text-to-Speech (TTS)\n- **Coqui TTS**: Advanced deep learning toolkit for high-quality, natural-sounding voice synthesis with support for British English models.\n\n## Wake-Word Detection\n- **Open-source (Vosk-based)**: A lightweight, local implementation for detecting the "Jarvis" wake-word without external dependencies.\n\n## Offline Operations\n- **Local Model Execution**: All AI/ML models (Whisper, Coqui TTS, Vosk) are downloaded and executed locally on the host machine.\n- **No External APIs**: No reliance on third-party cloud services or internet-connected APIs during operation.\n\n## Infrastructure & Tooling\n- **Dependency Management**: Standard `pip` and `requirements.txt`.\n- **Audio Processing**: `PyAudio` or `SoundFile` for capturing and playing back audio streams.\n- **Version Control**: Git (hosted on GitHub).
